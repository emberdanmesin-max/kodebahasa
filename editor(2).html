🔢 Variabel ▾
variabel = nilai

🔁 Perulangan ▾
untuk N kali lakukan:
Ulangi blok sebanyak N kali
untuk 5 kali lakukan: tuliskan("Halo!")
untuk X dari A sampai B lakukan:
Ulangi dengan variabel counter
untuk i dari 1 sampai 5 lakukan: tuliskan(i)
saat ... lakukan:
Ulangi selama kondisi benar
x = 0 saat x < 5 lakukan: tuliskan(x) x = x + 1
untuk setiap X dalam DAFTAR lakukan:
Iterasi elemen daftar
buah = ["apel", "mangga", "jeruk"] untuk setiap b dalam buah lakukan: tuliskan(b)
hentikan / lanjutkan
break / continue
❓ Kondisi ▾
jika ... maka:
Blok if
jika umur >= 17 maka: tuliskan("Boleh memilih")
atau jika ... maka:
Blok else if
atau jika umur >= 13 maka: tuliskan("Remaja")
jika tidak:
Blok else
jika tidak: tuliskan("Anak-anak")
